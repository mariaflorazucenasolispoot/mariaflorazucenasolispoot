# Actividades de Aprendizaje (ADAs) del Curso

Este archivo contiene todas las Actividades de Aprendizaje (ADAs) vistas en el curso.

## ADA 1: Introducción a la Programación
- **Descripción**: Conceptos básicos de programación y algoritmos
- **Objetivos**:
  - Entender qué es un algoritmo
  - Conocer los elementos básicos de programación
  - Identificar tipos de datos
- **Actividades**:
  - Creación de algoritmos simples
  - Ejercicios de pseudocódigo
  - Diagramas de flujo básicos

## ADA 2: Estructuras de Control
- **Descripción**: Estructuras condicionales y ciclos
- **Objetivos**:
  - Implementar estructuras if-else
  - Utilizar ciclos for y while
  - Aplicar control de flujo en programas
- **Actividades**:
  - Programas con condicionales
  - Ejercicios de ciclos
  - Resolución de problemas con iteraciones

## ADA 3: Funciones y Procedimientos
- **Descripción**: Modularización del código mediante funciones
- **Objetivos**:
  - Crear funciones reutilizables
  - Entender parámetros y valores de retorno
  - Aplicar el principio de modularidad
- **Actividades**:
  - Desarrollo de funciones matemáticas
  - Funciones con múltiples parámetros
  - Ejercicios de recursividad

## ADA 4: Estructuras de Datos
- **Descripción**: Arreglos, listas y estructuras básicas
- **Objetivos**:
  - Trabajar con arreglos unidimensionales y multidimensionales
  - Implementar listas
  - Manipular colecciones de datos
- **Actividades**:
  - Operaciones con arreglos
  - Búsqueda y ordenamiento
  - Ejercicios con matrices

## ADA 5: Programación Orientada a Objetos
- **Descripción**: Conceptos de POO y su implementación
- **Objetivos**:
  - Entender clases y objetos
  - Implementar encapsulamiento
  - Aplicar herencia y polimorfismo
- **Actividades**:
  - Creación de clases básicas
  - Ejercicios de herencia
  - Proyectos con POO

## ADA 6: Manejo de Archivos
- **Descripción**: Lectura y escritura de archivos
- **Objetivos**:
  - Leer datos desde archivos
  - Escribir información en archivos
  - Gestionar excepciones en operaciones de archivo
- **Actividades**:
  - Programas de lectura de texto
  - Almacenamiento de datos
  - Procesamiento de archivos CSV

## ADA 7: Algoritmos de Búsqueda y Ordenamiento
- **Descripción**: Implementación de algoritmos clásicos
- **Objetivos**:
  - Implementar búsqueda lineal y binaria
  - Programar algoritmos de ordenamiento
  - Analizar complejidad algorítmica
- **Actividades**:
  - Búsqueda en arreglos
  - Bubble sort, Selection sort
  - Comparación de eficiencia

## ADA 8: Estructuras de Datos Avanzadas
- **Descripción**: Pilas, colas y listas enlazadas
- **Objetivos**:
  - Implementar pilas (stacks)
  - Crear colas (queues)
  - Trabajar con listas enlazadas
- **Actividades**:
  - Operaciones con pilas
  - Simulaciones con colas
  - Implementación de listas enlazadas

## ADA 9: Recursividad
- **Descripción**: Técnicas de programación recursiva
- **Objetivos**:
  - Entender el concepto de recursividad
  - Implementar funciones recursivas
  - Resolver problemas con recursión
- **Actividades**:
  - Factorial y Fibonacci recursivos
  - Torres de Hanoi
  - Recorrido de estructuras recursivas

## ADA 10: Manejo de Excepciones
- **Descripción**: Gestión de errores y excepciones
- **Objetivos**:
  - Identificar tipos de errores
  - Implementar try-catch
  - Crear excepciones personalizadas
- **Actividades**:
  - Validación de datos
  - Control de errores en programas
  - Debugging y testing

## ADA 11: Bases de Datos
- **Descripción**: Introducción a bases de datos y SQL
- **Objetivos**:
  - Conectar con bases de datos
  - Ejecutar consultas SQL
  - Realizar operaciones CRUD
- **Actividades**:
  - Consultas SELECT básicas
  - INSERT, UPDATE, DELETE
  - Joins y relaciones

## ADA 12: Proyecto Final
- **Descripción**: Integración de todos los conceptos aprendidos
- **Objetivos**:
  - Aplicar todos los conceptos del curso
  - Desarrollar un proyecto completo
  - Documentar y presentar el trabajo
- **Actividades**:
  - Planificación del proyecto
  - Desarrollo e implementación
  - Presentación y documentación

---

## Notas Generales

Este documento sirve como referencia de todas las actividades de aprendizaje cubiertas durante el curso. Cada ADA está diseñada para construir sobre los conocimientos previos y preparar para las siguientes actividades.
